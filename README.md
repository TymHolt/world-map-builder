# World Map Builder
A game engine. Nothing special.

#### Required libraries (to build):
- [lwjgl](https://www.lwjgl.org/download)
- [joml](https://github.com/JOML-CI/JOML/releases)
- [bitmap-font-generator](https://github.com/TymHolt/bitmap-font-generator/releases)

LWJGL and JOML can be downloaded together using this [configurator](https://www.lwjgl.org/customize).

#### Flags
Debug mode and output
> -debug

Custom maximum updates per second
> -ups\[amount\]
