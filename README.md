# World Map Builder
A game engine. Nothing special.

#### Required libraries:
- [lwjgl-3.3.6](https://www.lwjgl.org/download)
- [joml-1.10.7](https://github.com/JOML-CI/JOML/releases)

LWJGL and JOML can be downloaded together using this [configurator](https://www.lwjgl.org/customize).

#### Flags
Debug mode and output
> -debug

Custom maximum updates per second
> -ups\[amount\]
