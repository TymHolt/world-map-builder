# World Map Builder
A tool to create basic 3D worlds.

Required libraries:
- lwjgl-3.3.6
- joml-1.10.7

Required tools:
- resource-bakery 0.2.0

#### Controls
- WASD - Move
- Arrows - Look around

#### Developing
If you change resources (Shader code, Images, etc.) run the bake.sh script
to bake the resources into the source code
